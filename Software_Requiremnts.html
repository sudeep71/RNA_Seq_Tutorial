Software Requirements¶

You will need access to a SHH client before the start of the class. It is important to install this before the lab class and to get familiar with it.

Mac and Linux Users:¶

Before we start certain house keeping stuff. Id you are using a Mac or a Linux laptop you have all the necessary software needed for this exercise.

Windows Users:¶

If you have Windows, you will need an SSH client. We suggest that you install mobaxterm, http://mobaxterm.mobatek.net/download.html - get the install edition and please try to do this before the class. The install is straight forward and if you need help, please contact before class.

Using the HPCC:¶

Please get familiar with the HPCC wiki commands found here (https://wiki.hpcc.msu.edu/display/hpccdocs/Documentation+and+User+Manual)
